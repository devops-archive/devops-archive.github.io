# DTI Wiki

This is the Devops Archive wiki page. This website serves are a documentation for commonly used Free and Open Source Softwares. Additionally we have documents on GNU/Linux OS Administration, Network Administration and Devops Automations.

This is small documentation website managed by a single person, errors can happen. While we sincerely try to keep the documentation as accurate as possible, please note there might be outdated data or incorrect details etc. Please kindly read the documents carefully and before using these documentation directly on production or live environments, test the same on lower environments like Dev or QA and confirm the compatibility and proof of working. Always make backups of the files and database before modifying.

Follow the rules, Stay safe.
